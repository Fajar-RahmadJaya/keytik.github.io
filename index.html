KeyTik: The All-in-One Automation Tool & Keyboard Remapper with Profiles

KeyTik is an open-source automation tool that can do almost all automation at your will. With AutoHotkey as scripting language, you can remap keys, create automation scripts, and more.

How To Install

Follow these steps to install KeyTik:

  1. Download AutoHotkey from here
  2. Download KeyTik zip file from here
  3. Extract the zip file
  4. Open the extracted folder and run KeyTik.exe
  5. Done!

What Makes Us Different Than Other Automation Tools

  • Uses AutoHotkey as the scripting language
  • Activate or deactivate profiles individually
  • Run profiles on startup
  • Text mode for editing AutoHotkey scripts directly
  • Lightweight and simple GUI

Features

  • Create, Edit, and Store Remap Profiles
  • Assign Shortcuts to Profiles
  • Run Profiles On Startup
  • Auto Clicker Integration
  • Pin Profiles for quick access

How To Use KeyTik As Auto Clicker

KeyTik comes with an integrated Auto Clicker. By default, holding the e key activates the Auto Clicker. You can customize the key and other settings as you like.

Example How You Can Use Each Feature

1. Multiple Profiles

Activate or deactivate remap profiles based on different conditions like games, work, etc.

2. Always on Top

Use this feature to keep KeyTik always on top for easy access during gaming or other tasks.

User Safety (For Those Worried About Viruses or Data Safety)

KeyTik is open-source, so you can be sure there are no hidden viruses. You can scan it using VirusTotal for further peace of mind.
